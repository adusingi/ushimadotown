# Ushimadotown Tourism Website

Official tourism website for Ushimado, Okayama Prefecture.

## Project Structure

- `/docs` - Project documentation
- `/public` - Static assets (images, CSS, JavaScript)
- `/src` - Source code
- `/tests` - Test files

- Node.js (v18 or higher)
